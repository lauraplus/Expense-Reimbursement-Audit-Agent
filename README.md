搭建 ReAct Agent，自动理解报销场景、按需检索政策条款、自主调用关联系统完成合规判定，在测试集上实现分钟级自动完成财务审核任务，审核准确率达85%+

**ReAct Agent、Agentic RAG、MCP Tool Use、Skill、向量检索、Human-in-the-Loop**

1. **ReAct 审核引擎**：Agent 接收报销单后，理解报销类目与事由描述，自主规划审核路径，通过 Thought-Action-Observation 循环动态决定查询哪些系统、检索哪些政策条款，输出带政策条款引用的审核结论
2. **MCP 动态工具调用**：通过 MCP 协议接入 HR 系统、预算系统、专项额度系统、报销历史和发票平台五个 MCP Server，Agent 根据报销类型自主选择工具组合——团建费用调用 HR 系统查询团队 base 地判断是否跨城、调用预算系统查询团建余额；AI 办公费用调用专项额度系统查询月度使用情况；招待费用调用报销历史查询月度累计
3. **政策知识库 RAG**：将费控政策文档按条款粒度 chunk 并 embedding 存储，Agent 在推理中按需检索相关条款，审核结论引用具体条款编号；政策更新仅需刷新知识库，无需修改 Agent 代码
4. **Skill 封装审核经验**：将团建费、差旅费、AI 办公费、招待费等高频类目的审核要点封装为 Skill，指导 Agent 在特定场景下关注正确的审核维度与工具调用组合
5. **人机协同与审计**：常规合规单据 Agent 自动通过，高金额和存疑单据强制转人工复核；完整推理链全量记录支持审计追溯；人工反馈闭环持续优化审核策略
